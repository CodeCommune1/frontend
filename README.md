## Installation

To install the dependencies, run:

```bash
npm install
```

## Running the Project

To start the development server, run:

```bash
npm run dev
```

## Building the Project

To build the project for production, run:

```bash
npm run build
```

## Linting

To lint the code, run:

```bash
npm run lint
```
